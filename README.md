# Sport Facility Booking System — Frontend

React + TypeScript client for the Sport Facility Booking System.

## Tech Stack
- React 18 + TypeScript
- Redux Toolkit + RTK Query
- Tailwind CSS
- React Hook Form + Zod
- react-i18next (PL + EN)
- Vitest + Cypress
